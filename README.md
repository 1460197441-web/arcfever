# 神经突触微信小程序

当前版本已收敛为“仪器共享交易平台”原型，重点围绕以下角色实现：

- 已认证个人用户
  - 输入姓名、电话
  - 从管理员维护的学校 / 学院列表中选择认证归属
  - 认证后既能发布自己的仪器，也能购买他人发布的仪器
- 管理者 / 平台方
  - 维护认证学校与学院
  - 查看待审核认证数量、平台仪器数量和纠纷概览
  - 承担审核和秩序维护角色

## 已实现内容

- 首页：角色说明、快捷入口、精选仪器、平台统计
- 分类与仪器列表：分类浏览、关键词搜索、仪器详情
- 交易链路：加入购物车、订单确认、支付页、支付成功页
- 用户中心：认证信息、我的仪器、我的订单、我的收藏、我的评价
- 发布链路：发布仪器、编辑仪器
- 沟通链路：会话列表、聊天详情
- 管理后台：后台概览、学校学院维护

## 技术实现

- 前端：原生微信小程序
- 后端：`cloudfunctions/gateway` 云函数
- 运行策略：
  - 优先调用云函数
  - 若你尚未配置云开发，前端会自动回退到本地 mock 数据

## 导入方式

1. 打开微信开发者工具
2. 选择“导入项目”
3. 选择项目目录：`C:\Users\arcfever\Documents\New project`
4. 如需直接体验，可先使用测试号或你自己的小程序 AppID

## 当前数据规则

- 认证表单只要求姓名、电话
- 学校与学院从管理员维护的数据中选择
- 支付、审核、聊天当前为原型演示数据流，便于后续接入真实后端
